# APZ-Tool – Einfache Zusammenfassung von Stahl-Zertifikaten

**Willkommen!**  
Dieses kleine Programm hilft dir, Abnahmeprüfzeugnisse (APZ) von Stahlblechen schnell in eine übersichtliche Excel-Tabelle umzuwandeln.  
Du brauchst **keine Programmierkenntnisse** – nur Doppelklick und ein paar Mausklicks.

### Was macht das Tool?
- Du wählst PDF-Dateien mit Stahl-Zertifikaten aus (z. B. von Dillinger, voestalpine oder Ilsenburger).
- Das Tool liest die wichtigsten Daten automatisch aus (Material, Dicke, Prüfwerte, Masse usw.).
- Es erstellt eine Excel-Datei mit einer klaren Tabelle – fertig zum Speichern oder Drucken.

**Beispiel**: Aus 5 PDF-Zertifikaten macht es in Sekunden eine Excel mit allen Blechen übersichtlich nebeneinander.

### Download & Start (super einfach – keine Installation!)

1. **Herunterladen**:
   - Gehe auf diese Seite: https://github.com/WojciechChomanski/APZ-Tool/releases
   - Lade die Datei `APZ-Tool.exe` herunter (klicke auf den Link, speichere sie z. B. auf deinen Desktop).

2. **Starten**:
   - Gehe zum Ordner, wo du die Datei gespeichert hast (z. B. Desktop).
   - **Doppelklick** auf `APZ-Tool.exe`.
   - Ein Fenster öffnet sich – das ist das Programm!

   **Tipp**: Rechtsklick auf die Datei > "Verknüpfung erstellen" > Ziehe die Verknüpfung auf den Desktop. So startest du es immer mit einem Klick.

**Keine Installation nötig** – das Programm läuft direkt. Es speichert nichts auf deinem Computer außer der Excel-Datei, die du selbst anlegst.

### So benutzt du das Tool

1. Fenster öffnet sich.
2. Klicke auf **"PDFs auswählen"**.
3. Wähle deine PDF-Zertifikate aus (du kannst mehrere mit Strg-Taste markieren).
4. Optional: Ändere den Namen der Excel-Datei (standard: APZ_Zusammenfassung_2025.xlsx).
5. Klicke auf **"Start"**.
6. Warte kurz (Fortschrittsbalken zeigt an).
7. Fertig! Die Excel-Datei wird im selben Ordner gespeichert wie die .exe.

**Beispiel-Ergebnis**: Eine Tabelle mit Spalten wie Material, Dicke, Prüfwerte (alles "ok" oder "entf."), Masse usw. – alles automatisch aus deinen PDFs.

### Programm löschen (Deinstallieren)

Das Tool ist **portabel** – es installiert nichts im Hintergrund.

- Zum Löschen: Rechtsklick auf `APZ-Tool.exe` > "Löschen".
- Wenn du eine Verknüpfung auf dem Desktop hast: Auch diese löschen.
- Fertig – keine Spuren bleiben zurück.

Du kannst es jederzeit neu herunterladen und wieder benutzen.

### Probleme? Hier die häufigsten Lösungen

- **Fenster öffnet sich nicht**: Rechtsklick auf die .exe > "Als Administrator ausführen" probieren.
- **Antivirus meldet etwas**: Das Tool ist sicher (open-source Code). Füge es zur Ausnahmeliste hinzu.
- **Excel wird nicht erstellt**: Stelle sicher, dass du PDFs ausgewählt hast.
- **Fragen?** Schreib mir eine Nachricht auf GitHub (Issues) oder per E-Mail (im Repo steht meine Adresse).
